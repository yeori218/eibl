Overview

누구나 사용할 수 있는
간편한 내 돈 관리 앱

귀찮아서 쓰지 않던 가계부, 어렵게만 느껴지는 재테크 및 금융상품
기존에 다소 무겁고 어려운 금융서비스보다 편리하고 가까운 서비스를 제공하기 위해
누구나 사용할 수 있다는 뜻의 에이블 어플을 기획하였습니다,
깔끔한 레이아웃과 직관적인 아이콘사용으로 접근성을 최대화하였습니다.

그에 맞는 무드보드를 일러스트로 표현하였습니다.

Wireframe

User Flow

사용성 분석에 의한 유저 플로우

Interface

01. Walk through

에이블의 포인트 색상을 이용해 스플래쉬와 로그인 페이지를 제작하고
워크스루 페이지를 제작하여 앱의 주요 서비스와 내용을 사용자들에게 명확하고 간결하게 전달합니다.

02. Main

한눈에 보는 내 모든 자산현황

메인 홈화면에서 나의 자산, 신용 현황을 한눈에 볼 수 있고
각 계좌별로 상세 내역을 확인 할 수 있으며 카드별 사용내역을 파악 할 수 있습니다,
또한 투자, 대출, 보험 등 모든 금융정보를 보여주며 홈 편집 기능으로
사용자가 원하는 정보만을 표시 할 수 있습니다.

03. All menu & Edit

모든 메뉴를 한눈에 보여주는 전체 메뉴와
내게 맞춤 홈 편집 / 금액 숨기기 기능까지

04. Sub1 - Household Accounts

언제 어디서나 간편한
맞춤 가계부

사용자가 직접 예산을 설정해 관리 할 수 있으며
핵심 지출내역들을 한눈에 볼 수 있습니다
카테고리별 예산을 설정하고 나에게 필요한 카테고리만 편집도 가능합니다

05. Sub2 - Recommend

인기상품 추천은 물론
나에게 맞춤 추천까지

최신 금융상품 트렌드와 사용자의 소비 패턴 및
자산현황과 신용등급을 고려해
딱 맞는 상품과 혜택을 찾아드립니다

06. Sub3 - Setting

개인설정과 연동관리로 편리한 앱 관리
